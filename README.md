データベース内のテーブルは5つ
1．POSTテーブル
2．commentテーブル　- どのポストのテーブルかをpostIDで紐づける
3. USERテーブル　- どのポスト、コメントかをuserIDで紐づける
4．【NEW!】USER情報テーブル
5．【NEW!】お気に入りテーブル　- SESSIONのuserがおきに入り登録したポストを保存する

追加機能
1．プロフィール画像を動的に変更できる（javascript）
2．プロフィール画像がコメントに表示される
3．お気に入り登録をし、プロフィールに保存ポストが表示される
